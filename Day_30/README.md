The User Management System is a simple full-stack web application built using React (with Vite) for the frontend and Express.js with Node.js for the backend. It follows a REST API architecture and uses CORS and JSON middleware to enable smooth communication between the client and server. The application allows users to view all existing users, add new users, and delete users, with data stored in memory on the backend. To run the project, clone the repository, install the required dependencies for both the frontend and backend, start the backend server, and then run the frontend using `npm run dev`.
Learning Outcomes

REST API Development

Handling HTTP Methods (GET, POST, DELETE)

React State Management

Full-Stack Integration

Debugging Port & CORS Issues
